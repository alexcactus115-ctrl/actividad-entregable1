# Analizador de Arreglo

## Autor: Alejandro Noel Bello Sanchez

## Descripción
Este proyecto analiza un arreglo de números enteros y realiza dos tareas:
1. Verifica si el número máximo se repite más de una vez.
2. Calcula el promedio del arreglo y lo evalúa como "BIEN" si es >= 5 o "MAL" si es < 5.
